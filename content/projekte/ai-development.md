**AI-Development** ist mein persönlicher Workflow für KI-gestützte Softwareentwicklung: eine Kombination aus librechat und opencode, die mir erlaubt, Projekte effizient zu planen, zu entwickeln und zu deployen.

## Workflow

1. **Idee & Planung** im Dialog mit einem LLM (librechat)
2. **Iterative Entwicklung** mit opencode — das LLM arbeitet direkt im Repository
3. **Deployment** via GitHub Webhook – Push auf main baut die Seite neu
4. **Lessons Learned** werden in agents.md dokumentiert

## Entstehung

Diese gesamte Website ist mit diesem Workflow entstanden – das Projekt dokumentiert also sich selbst.
