**html2md-flask** ist eine Flask-Webapp zur Extraktion von Webseiten-Hauptinhalten als Markdown.

## Features

- URL eingeben, Markdown-Vorschau erhalten
- Extrahiert nur den Hauptinhalt (keine Navigation, kein Footer)
- Download als .md-Datei
- Docker-ready mit docker-compose
- Automatischer Docker-Build via GitHub Actions

## Entstehung

Gedacht für Redakteure und Autoren, die schnell Inhalte von Webseiten in Markdown konvertieren müssen – ohne Kommandozeile.
